# Phase 2 Notes — Public Frontend Website

This phase contains only the customer-facing public website for Touch Threading and Brows Salon.

Included:
- React + Vite frontend
- Tailwind CSS theme
- Home page
- Services/prices page
- Offers page
- Appointment booking page
- Staff page
- About page
- Gallery page
- Reviews page
- Contact page
- Reusable public components
- Public API client wired to Phase 1 backend

Not included yet:
- Admin dashboard
- Admin login UI
- README/setup/deployment docs
- Final seed/deployment verification

Those belong to Phase 3 and Phase 4.
